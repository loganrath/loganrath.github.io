📋 How to connect your Google Sheet
  1. Create a new Google Sheet. Name the columns in row 1: name, formula, charge (optional), notes (optional).
  2. Fill in each row with one polyatomic ion — e.g., Ammonium | NH₄ | +1.
  3. Share the sheet: click File → Share → Share with others, then choose "Anyone with the link" and set access to Viewer.
  4. Go to File → Share → Publish to web. Choose Sheet 1 and Comma-separated values (.csv), then click Publish.
  5. Copy the spreadsheet ID from your sheet's URL:
    https://docs.google.com/spreadsheets/d/<SPREADSHEET_ID>/edit
  6. Paste the ID into the Spreadsheet ID field below and click Load Sheet.
  7. If you have multiple tabs, also paste the numeric Sheet GID from the URL's #gid=… hash (default is 0).

Column format: name (required) · formula (required) · charge (optional) · notes (optional). The first row is treated as a header and is skipped.

Using built-in ion list. Paste a Sheet ID and click Load to override.
