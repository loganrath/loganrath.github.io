Drag & Drop Keyword Builder

Stepwise: question → drag words → add synonyms → final Boolean output on its own screen.

Step 1
Enter research question
Step 2
Drag single words into keyword boxes
Each token is a single word from your question. Drag and drop into a box.
Step 3
Add synonyms (no Boolean shown here)
Add synonyms for each selected keyword (press Enter or comma to add). The Boolean result is on the next screen.
You won't see the final Boolean string on this screen. Use the "View Boolean" button to go to the final screen where you can copy it.
Current keywords (editable) and synonyms are being saved — proceed to the final step to view the generated Boolean string.
Step 4
Boolean output & copy
Your Boolean string will appear here.
Paste this string into your library database search box. Adjust quotes/field tags as needed for specific databases.
Tip: If a keyword should be multiple words, drop several words into the same box in Step 2 and edit it in Step 3.
